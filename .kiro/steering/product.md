# PlanDaily - Product Overview

PlanDaily is a modern React-based daily planner and goal management application designed to help users organize their tasks, track goals, and manage their time effectively.

## Core Features

**User Features:**
- Daily task planning with priorities, tags, and reminders
- Goal management with milestone tracking and progress visualization
- Calendar view with color-coded events and categories
- Comprehensive settings for profile, notifications, and security
- Task organization with descriptions, due dates, and custom tags

**Admin Features:**
- Dashboard with user statistics and activity metrics
- User management with role and status controls
- Detailed user profiles with activity tracking
- Data export functionality (CSV, Excel, PDF)

## Target Users
- Individuals seeking better personal organization and productivity
- Users who want to track and achieve personal/professional goals
- People who prefer visual calendar-based planning
- Administrators managing user accounts and platform analytics

## Key Value Propositions
- Unified interface combining daily planning, goal tracking, and calendar management
- Vietnamese language support with localized content
- Mobile-responsive design for cross-device usage
- Admin dashboard for platform management and user insights
